Story


Sally is a mobile friendly dating website powered by Angular, ASP Net, and Azure Cloud Services.

Users of Sally can:

  1. Have a detailed Profile
  2. Upload, make Main, and Delete Photos
  3. Determine matches based on gender and age preferences
  4. Like other users, and have a list of liked users and likers
  5. Message Users - Users have an inbox, outbox, and logs for all users

Register now and find your next match with Sally!
